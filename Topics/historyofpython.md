# History Of Python

So let's begin the series of learning python. The first topic is history of python. We need to know at least the history of the language we are going to learn isn't it?

- Designed by [Guido Van Rossum](https://en.wikipedia.org/wiki/Guido_van_Rossum?wprov=sfla1)
- Started working on this language in 1989 in Netherlands.
- Guido was working in [CWI](https://en.wikipedia.org/wiki/Centrum_Wiskunde_%26_Informatica) at that time in a distributed OS [Amoeba](https://en.wikipedia.org/wiki/Amoeba_%28operating_system%29).
- He had on [ABC](https://en.wikipedia.org/wiki/ABC_%28programming_language%29) programming language before in early 1985.
- It was inspired from ABC programming language also we can say successor of ABC programming language.
- During Christmas holiday he started working on the idea as a hobby project. So he took features of ABC and fix the flaws which was in ABC language.

> Take the best and fix the rest!

- So open working for 3 months on it, he came up with  new scripting the  language named "[Python](https://en.wikipedia.org/wiki/Python_%28programming_language%29?wprov=sfla1)".
- The name is taken from "[Monty Python's Flying Circus](https://en.wikipedia.org/wiki/Monty_Python%27s_Flying_Circus)".
## Versions
- First released in February 1991 (version 0.9.0)
> The primary discussion forum for Python was formed.
- Official version 1.0 released in January 1994 with new features.
- Version 2.0 in 2000
- Version 3.0 in 2008

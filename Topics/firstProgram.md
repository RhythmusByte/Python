## Writing our First Program

In this section, we'll explore how to write our initial program in Python. To print something, we employ the `print()` function in Python. Inside the parentheses, we must provide the content to be printed, enclosed within single or double quotes. For instance, to display the classic "Hello World!" message, we simply type:

```
print('Hello World!')
```

This will output `Hello World!`.

> If you omit passing any arguments, it's acceptable, but you still need to include empty parentheses at the end to instruct Python to execute the function. Consequently, `print()` will generate an invisible newline character, resulting in a blank line appearing on your screen.

Thus, `print()` is a function utilized for printing content. We can adopt either of the following two approaches to achieve the same outcome:

```
print('Hello World!') 
```
or
```
print("Hello World!")
```

Additionally, it's worth noting that both single and double quotes are permissible in Python.
